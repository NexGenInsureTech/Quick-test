    1. Predictive Model

    Score = (1 - Penetration)*0.4 + (CASA scaled)*0.3 + (1 - Digital)*0.3

0.4 → Penetration (highest impact)
0.3 → CASA (customer base)
0.3 → Digital gap (efficiency)

These are heuristic weights → to be calibrated later

    2. COA Model

    Base Premium = SUM(Premium_Collected)

Elasticity = %Δ Premium / %Δ COA

Premium = Base × (1 + COA × Elasticity)

Elasticity must be derived from:
- A/B testing
- Historical campaigns
- Regression

    3. Campaign Engine

    Low Penetration + High CASA:
→ Cross-sell campaign
→ Target CASA customers
→ Product: PA + Health
→ Channel: Branch + SMS

Low Digital:
→ Digital training + app push

High Loss Ratio:
→ Reduce risk exposure

    4. Monitoring

    Conversion = Policies / Opportunities
COA per policy = Cost / Policies

Interpretation:
High conversion + low COA = efficient
Low conversion + high COA = waste

COA Optimization (Deep Logic)

BASE PREMIUM:
Derived from uploaded dataset = sum of Premium_Collected

ELASTICITY CONCEPT:
Elasticity = % change in output / % change in input

Here:
Elasticity = % change in premium / % change in COA

Example:
If COA +10% → Premium +5% → Elasticity = 0.5

IMPORTANT:
- Elasticity varies by bank, product, region
- Should be calculated using historical campaigns

RECOMMENDED METHOD:
Run regression:
Premium Growth ~ Incentive Spend

OR
A/B Testing:
Group A: Normal COA
Group B: Increased COA

Measure difference → derive real elasticity
